# задание 1.1 

### cколько записей вернет запрос select * from Table_A inner join Table_B?

так как запрос без условия, то будет выполнено декаартово произведение обеих таблиц - каждая строка первой таблицы будет обьединена с каждой строкой второй таблицы.

### cколько записей вернет следующий запрос? select * from Table_A a inner join Table_B b on a.ID=b.Id_A

теперь есть условие, поэтому количество записей будет равно количеству строк, в который выполняется условие a.ID=b.Id_A

### чем отличаются следующие запросы?
### select * from Table_A a inner join Table_B b on a.ID=b.Id_A
### select * from Table_A a left join Table_B b on a.ID=b.Id_A

inner join возвращает совпадения в обеих таблицах, а left join всегда выведет все значения Table_A, и совпадающие B, но если B нету подходящих, то выведет вместо значения NULL
